Nintendo Countdown Calendar

The Nintendo Countdown Calendar app is a DSiWare calendar app that can be used to view the countdown for events. It was priced at 200 DSi Points.

Summary

The Countdown Calendar starts off with setup dialog for the user's birthday and name, which are used for the birthday event. Events can repeat annually, monthly, weekly, etc. and are paired with a written title with the stylus and an image from a library of 132 icons. When revisiting the app, it will show the user how many days are left until the closet event(s) happen.
