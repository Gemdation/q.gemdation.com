In this folder you'll find the WiimmfiDNS application.

That is a small application that you can run on your computer, which will then run a custom DNS server for Wiimmfi. This is useful if your router or your ISP is blocking custom DNS servers. For more information (especially if you're unsure which of these versions you need), see this page.
