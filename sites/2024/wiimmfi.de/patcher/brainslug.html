Wiimmfi Brainslug Module

Overview

Brainslug is a software created by Chadderz that can be used to easily add code modifications to existing games. You can find out more about Brainslug on its GitHub page.

If you are a developer and you want to use Wiimmfi while using the Brainslug channel, you can download this module.

Download and setup

The newest version of the Wiimmfi brainslug module can be downloaded here:

  • Wiimmfi Brainslug Module (zip, 8 KB)

Extract the ZIP archive to your SD card, then start the Brainslug Channel from the Homebrew Channel.
