Auto Wiiware Patcher

Overview

The Auto Wiiware Patcher has been created by the RiiConnect24 team. It can be used to take existing WAD files of a WiiWare game, and patch these files so the game connects to Wiimmfi. For more information see the Github page.

Download and setup

Download the newest version of the Auto Wiiware Patcher from here and extract it. Then take the WAD files you'd like to patch, put them into the folder corresponding to your operating system, and then run the patcher.sh file (Linux/Mac) or patcher.bat file (Windows).

Follow the instructions and the patcher will create a new WAD file that you can install on your Wii with a WAD Manager.
