Privacy Policy

Wiimmfi consists of 2 components: This website and the service where Nintendo's game consoles Wii and DS can log in. Because the website and the service are closely linked and the website displays data about the service, there is a common privacy policy statement.

Cookies

This website uses a single session cookie (WIIMMFI_SESSION) for technical reasons only.

This cookie is used to remember filter settings and pager scroll positions on pages that are using the pager system, like the ban list.

For logged-in users, this cookie is additionally also used to keep track of the currently logged-in session, until the user logs out again.

The data for the session is archived in a database. For guests, these sessions are automatically deleted no later than 20 days after the last visit. The timeout for logged-in users is 100 days. Deleting the session invalidates the cookie and makes it unusable.

In addition to this Wiimmfi Cookie, the website also uses technical cookies from Cloudflare which are needed to protect our website from attacks. For more information see the "Cloudflare" section below.

If you do not want to use cookies, you can either instruct your browser to not accept cookies from this website, or you avoid visiting this website. The single cookie Wiimmfi uses is needed for technical purposes, so there's no builtin feature of the Wiimmfi webpage to prevent setting this cookie.

JavaScript

This website uses JavaScript. No JavaScript content is included from external webpages, all scripts are loaded directly from wiimmfi.de or one of its subdomains. The only exception is the access control scripts by CloudFlare (see below).

If you don't want to use JavaScript on Wiimmfi you can disable it in your browser, but that will break various features of the Wiimmfi webpage.

JavaScript is not used for any kind of tracking, it is only used for various display-related functions, for example the custom pager used to scroll through different lists like the ban list, for the background color bars in the login statistics, or to prevent robots from endlessly crawling through the various webpages by making some links only accessible through JavaScript. It's also used for security reasons (Cloudflare).

CloudFlare

In order to protect the webpage from attacks and unauthorized access, traffic is routed through CloudFlare proxies. This traffic is then analyzed by CloudFlare to detect and block malicious requests. This requires the usage of certain cookies.

CloudFlare is a product by Cloudflare, Inc., 101 Townsend St, San Francisco, CA 94107, USA. CloudFlare operates a Content Delivery Network (CDN) and provides Firewall-like functionality for this website.

For more information please check the Cloudflare Privacy Policy.

Resources by third parties

This website does not use any resources or cookies from other websites (except for Cloudflare for security reasons) by default. This prevents tracking by third parties.

Some pages on this website may embed a YouTube video. Loading such a webpage does not send any data to YouTube or Google servers, the preview image is loaded directly from Wiimmfi. Only when you click on such a video to play it, multiple requests are getting sent to YouTube servers, just like it would happen when you watched a video directly on YouTube. This will also cause JavaScript code to be downloaded and executed from the YouTube servers.

The company YouTube LLC is owned by Google LLC. You can read their Privacy Policy on their website. Wiimmfi has no influence on what data YouTube and Google are storing once you play one of the embedded videos.

User accounts of Wiimmfi

Wiimmfi does not manage its own user accounts, but uses the user accounts of the Wii-Homebrew Forums. To log in, users must have a valid and active account there. Login name and password are then compared to the forum via a so-called bridge. This login is not tracked by the forum.

The only data transferred from the forum is the username. Further information is not necessary for the operation of Wiimmfi and will not be taken from the forum.

The public will not see the usernames. Wiimmfi also provides no communication options for users among each other. Therefore, the forum is used for all communication, which also supports private messages. This communication channel should also be used when contacting the Wiimmfi team.

Miscellaneous

No personal data is collected by this web site. The usual log files are only used in the case of security and regularly deleted with a time delay. A transfer of the data to third parties outside this security is excluded.

For statistic purposes, special log files are created containing only the time and web page and a few more statistical data. IP address, used browser or similar information about single visitors is not stored.

The Wiimmfi service

Nintendo discontinued its online service WFC in May 2014. Wiimmfi has jumped in the breach and provides a replacement service for free. For this, the games must be patched to use the the new server. The data protocol had to be taken over unchanged with all its weaknesses.

When logging in to the Wiimmfi service, the console (Nintendo Wii or DS) provides data used to administer, ban, and display real-time statistics on the website. The storage and control of the data is necessary for the technical process of the Wiimmfi service. In particular, these data are used to improve Wiimmfi, as not all details of the protocol are known.

The web page publicly displays the player names transmitted by the console. Therefore, no real name should be used. Usually, statistics about the games are available for up to 3 days.

Automatic linking of game data and the user account is not technically possible. However, any logged-in user of the Wiimmfi website can manually link consoles to his account. The link is only visible to the logged in user and moderators.

Data is collected for various statistics that go beyond years. Most of the statistics are published on the website while preserving the personal anonymity. The long-term statistics are only counters, e.g. the number of logins per period counts, but not the time of login.

For continuous improvement of the service, at times and without notice, significant logs will be created on the data provided by the game console. These logs are activated and evaluated only when needed and depending on the game. After the final evaluation they will be deleted.

If you decline the described processing of the data of your console, then you must use another alternative service for Nintendo's WFC!

Moderators and Bans

Moderators have access to the game data for up to 4 weeks, to resolve conflicts with hindsight. They can ban consoles on different kinds. A ban can be temporary or permanent. Permanent bans are usually used on repeated offenses. For this, the data of each individual ban must be archived permanently.

A note about accessing third-party servers

Some games are programmed by the game makers that they access other servers managed by third parties. Without these third-party servers, the games would not work anymore or only partially work. Wiimmfi has no influence on the behavior of the games and also not on the operators of the third-party servers or their privacy policy.

Datenschutz-Erklärung

Wiimmfi besteht aus 2 Komponenten: Dieser Webseite und dem Dienst, an dem sich Nintendos Spielkonsolen Wii und DS anmelden können. Da die Webseite und der Dienst eng miteinander verknüpft sind und die Webseite Daten des Dienstes anzeigt, gibt es eine gemeinsame Datenschutz-Erklärung.

Cookies

Diese Webseite benutzt ein einziges Sitzungs-Cookie (WIIMMFI_SESSION), welches nur aus technisch erforderlichen Gründen eingesetzt wird.

Dieses Cookie wird benutzt, um Filtereinstellungen und Scroll-Positionen zu speichern, wenn eine Seite das Pager-System verwendet, wie beispielsweise die Bann-Liste.

Für angemeldete Benutzer wird das Cookie außerdem dazu benutzt, die aktuelle Sitzung zu verfolgen. Dies geschieht so lange, bis der Nutzer sich wieder abmeldet.

Daten, die mit einer Sitzung verknüpft sind, werden in einer Datenbank gespeichert. Diese Sitzungsdaten werden für nicht angemeldete Benutzer ("Gäste") spätestens 20 Tage nach dem letzten Seitenbesuch gelöscht; für angemeldete Benutzer spätestens 100 Tage nach dem letzten Besuch. Diese Löschung macht das Cookie unbrauchbar.

Zusätzlich zu diesem Wiimmfi-Cookie werden auch technische Cookies von Cloudflare genutzt, die benötigt werden, um unsere Webseite vor Angriffen zu schützen. Weitere Informationen dazu stehen im "Cloudflare"-Absatz weiter unten.

Wenn sie nicht möchten, dass Wiimmfi Cookies in ihrem Browser anlegt, dann können sie entweder ihren Browser so konfigurieren, dass er keine Cookies annimmt (damit werden allerdings Teile der Wiimmfi-Seite nicht mehr funktionieren); oder sie verlassen diese Webseite. Da das einzige von Wiimmfi genutzte Cookie technisch zwingend erforderlich ist, gibt es keinen Mechanismus auf der Wiimmfi-Seite, das Setzen dieses Cookies zu verhindern.

JavaScript

Diese Webseite benutzt JavaScript. Es werden allerdings keine JavaScript-Inhalte von externen Webseiten eingebunden; alle Scripte werden direkt von wiimmfi.de oder einer seiner Subdomains geladen. Einzige Ausnahme davon sind die Sicherheitsscripte von CloudFlare (siehe unten).

Wenn sie JavaScript nicht benutzen möchten, dann können sie dieses in ihrem Browser deaktivieren. Damit werden allerdings einzelne Funktionen der Wiimmfi-Webseite nicht mehr funktionieren.

JavaScript wird nicht für Tracking irgendeiner Art benutzt, es wird nur für verschiedene Anzeigefunktionen genutzt. Dazu gehören unter anderem der Pager, der verwendet wird um durch Listen zu blättern (beispielsweise die Bann-Liste), die farbigen Hintergrundmarkierungen in den Login-Statistiken, oder zur Einschränkung von automatisierten Webseitenzugriffen durch sogenannte Robots, indem gewisse Links nur durch JavaScript erreichbar gemacht werden.

CloudFlare

Um unsere Webseite vor Angriffen zu schützen, wird der Traffic über CloudFlare-Proxies geroutet. Dieser Traffic wird dann von CloudFlare analysiert, um Angriffe und andere böswillige Zugriffe zu blockieren. Diese Überprüfung benötigt dabei gewisse Cookies.

CloudFlare ist ein Produkt von Cloudflare, Inc., 101 Townsend St, San Francisco, CA 94107, USA. CloudFlare betreibt ein sogenanntes Content Delivery Network (CDN) und stellt firewallähnliche Funktionalitäten für diese Webseite bereit.

Für weitere Informationen lesen sie bitte die Datenschutzerklärung von Cloudflare.

Einbindung von fremden Resourcen

Diese Webseite verwendet standardmäßig keine Resourcen oder Cookies von anderen Webseiten, mit Ausnahme der CloudFlare-Scripte. Damit wird ein Tracking durch Dritte unterbunden.

Auf einigen Unterseiten sind evtl. YouTube-Videos eingebunden. Das reine Aufrufen einer solchen Seite sorgt nicht dafür, dass Daten an YouTube- oder an Google-Server gesendet werden. Auch das Vorschaubild wird direkt von Wiimmfi geladen. Erst wenn sie ein solches Video anklicken, um es abzuspielen, werden Requests an YouTube-Server gesendet, genau so als würden sie dieses Video direkt auf der YouTube-Seite abspielen. Dabei werden auch JavaScript und andere Inhalte vom YouTube-Server heruntergeladen und ausgeführt.

Die Firma YouTube LLC ist im Besitz von Google LLC. Sie können deren Datenschutzerklärung auf Google's Webseite lesen. Wiimmfi hat keinen Einfluss darauf, welche Daten von Google oder YouTube erfasst werden, sobald sie ein solches Video abspielen.

Benutzerkonten von Wiimmfi

Wiimmfi verwaltet keine eigenen Benutzerkonten, sondern verwendet die Benutzerkonten des Wii-Homebrew-Forums. Zur Anmeldung müssen die Benutzer dort ein gültiges und aktives Konto besitzen. Anmeldename und Kennwort werden dann über eine so genannte Brücke mit dem Forum abgeglichen. Dabei werden im Forum keinerlei Spuren hinterlassen (kein Tracking).

Die einzigen Daten, die aus dem Forum übertragen werden, ist der Benutzername. Weitere Informationen sind für den Betrieb von Wiimmfi nicht nötig und werden auch nicht aus dem Forum übernommen.

Der Öffentlichkeit werden die Benutzernamen nicht angezeigt. Wiimmfi stellt auch keine Kommunikations­möglichkeiten für die Nutzer untereinander zur Verfügung. Daher wird für sämtliche Kommunikation das Forum genutzt, welches auch private Nachrichten unterstützt. Dieser Kommunikationsweg ist auch bei Kontaktaufnahme zum Wiimmfi-Team zu verwenden.

Verschiedenes

Es werden keinerlei persönliche Daten von dieser Webseite erhoben. Die üblichen Log-Dateien werden nur in Falle einer Gefahrenabwehr herangezogen und regelmäßig mit zeitlichem Abstand gelöscht. Eine Weitergabe der Daten an Dritte außerhalb dieser Gefahrenabwehr ist ausgeschlossen.

Für statistische Zwecke werden spezielle Log-Dateien erstellt, die nur den Zeitpunkt und die besuchte Seite sowie ein paar weitere statistische Daten enthalten. IP-Adresse, verwendeter Browser oder ähnliche Angaben über einzelne Besucher werden hierbei nicht gespeichert.

Der Wiimmfi-Dienst

Nintendo hat seinen Online-Dienst WFC im Mai 2014 eingestellt. Wiimmfi ist in die Bresche gesprungen und stellt einen Ersatzdienst kostenlos zur Verfügung. Zur Nutzung müssen die Spiele an den neuen Server angepasst werden. Das Datenprotokoll musste unverändert mit all seinen Schwächen übernommen werden.

Bei der Anmeldung am Wiimmfi-Dienst stellt die Konsole (Nintendo Wii oder DS) Daten zur Verfügung, die zur Verwaltung, dem Bann-System und zur Anzeige der Echtzeit-Statistiken auf der Webseite genutzt werden. Die Speicherung und Kontrolle der Daten ist für den technischen Ablauf des Dienstes notwendig. Insbesondere werden diese Daten zur Verbesserung von Wiimmfi genutzt, da noch nicht alle Details des Protokolls bekannt sind.

Die Webseite zeigt die Spieler-Namen, die von der Konsole übermittelt werden, öffentlich an. Daher sollten keine realen Namen verwendet werden. Üblicherweise stehen Statistiken über die Spiele bis zu 3 Tage zur Verfügung.

Eine automatische Verknüpfung von Spieldaten und dem Benutzer-Konto ist technisch nicht möglich. Allerdings kann jeder angemeldete Nutzer der Wiimmfi-Webseite Konsolen mit seinem Konto manuell verknüpfen. Die Verknüpfung ist nur für den angemeldeten Nutzer und für die Moderatoren sichtbar.

Es werden Daten für diverse Statistiken erhoben, die über Jahre hinausgehen. Die meisten Statistiken werden auf der Webseite unter Wahrung der Annonymität der einzelnen Personen veröffentlicht. Bei den Langzeitstatistiken handelt es sich nur um Zähler, die z.B. die Anzahl der Anmeldungen pro Zeitraum zählen, nicht aber den Zeitpunkt der Anmeldung.

Zur fortlaufenden Verbesserung des Dienstes werden zeitweise und ohne Ankündigung erhebliche Logs über die Daten, die die Spielkonsole liefert, angelegt. Diese Logs werden aber nur bei Bedarf und spielabhängig aktiviert und ausgewertet. Nach der abschließenden Auswertung werden sie gelöscht.

Sollten Sie die beschriebene Verarbeitung der Daten ihrer Konsole ablehnen, dann müssen sie einen anderen Ersatzdienst für Nintendos WFC verwenden!

Moderatoren und Banns

Moderatoren haben bis zu 4 Wochen Zugriff auf die Spieldaten, um auch im Nachhinein Konflikte zu lösen. Sie können Konsolen auf verschiedene Art vom Spielbetrieb ausschließen (bannen). Ein Bann kann zeitlich begrenzt oder permanent sein. Permanente Banns werden üblicherweise bei wiederholten Vergehen ausgesprochen. Hierfür müssen die Daten der einzelnen Banns dauerhaft archiviert werden.

Ein Hinweis über den Zugriff auf Dritt-Server

Manche Spiele sind von den Spieleherstellern so programmiert, dass sie auf weitere Server zugreifen, die von Dritten verwaltet werden. Ohne diese Dritt-Server würden die Spiele nicht mehr oder nur eingeschränkt funktionieren. Wiimmfi hat weder Einfluss auf das Verhalten der Spiele noch auf die Betreiber der Dritt-Server oder deren Datenschutz.
