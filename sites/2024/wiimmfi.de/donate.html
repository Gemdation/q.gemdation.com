Donate to the Wiimmfi Team

Introduction

In the past, we (Wiimm and Leseratte) have been asked multiple times by different people if there was a way to donate for Wiimmfi. We've always refused to accept donations in the past, as we didn't have any costs associated with operating Wiimmfi and didn't want to make money with it.

However, since Wiimmfi and Wii-Homebrew.com moved to a new server in November 2019, the server costs are no longer paid for by Quali (the owner of the Wii-Homebrew.com forum), but by Wiimm and Leseratte (the creators and admins of Wiimmfi).

Thus, we have decided to provide a way for people to donate to help pay for the Wiimmfi server costs. You are in no way obligated to donate in order to use Wiimmfi, Wiimmfi will still be freely usable by everyone, donations are completely voluntary. But if you do want to support us, you now have the ability to do so.

How to donate

We have set up a Patreon account, which allows you to set up a monthly donation for any amount you want. You can find the Patreon page by clicking on this link.

If you do not like to make a monthly contribution but would rather make a one-time donation using PayPal, you can click on this link to get to the Wii-Homebrew.com forum donation page.

Both the donations on the Patreon page and on the Wii-Homebrew.com forum donation page will be used to pay for the server costs for the physical server that Wiimmfi and Wii-Homebrew.com are hosted on.

  • Become a Patron (monthly donation on Patreon, PayPal or credit card)
  • Donate on Wii-Homebrew.com (one-time donation, PayPal)
