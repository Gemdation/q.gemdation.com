Wiimmfi on Dolphin

Overview

Dolphin is an open-source Wii and Gamecube emulator. You can connect to Wiimmfi using Dolphin, but you will need to import a NAND backup from a real Wii or WiiU console to do so.

Setup

In the past, there have been lots of cheaters using Dolphin who were able to get around our bans using auto-generated NAND backups. This is no longer possible, you need a NAND from a real console to play online with Dolphin. NANDs generated by tools like the "Wiimmfi Unbanner" will not work.

To create a NAND backup, you can use BootMii if you're on a Wii, or WiiU NAND Dumper if you're on a WiiU

Once you've done that, you should have a nand.bin and a keys.bin file. Make sure you're using an up-to-date version of Dolphin (5.0-stable is too old!), then go to Tools -> Manage NAND -> Import BootMii NAND Backup, and then select the nand.bin file from your SD card.

More information about importing a Wii NAND into Dolphin can be found on the NAND Usage Guide in the Dolphin Wiki.

After that, your Dolphin will be set up correctly and all that's needed is to patch your game image for Wiimmfi. Check out the tutorial for the ISO patcher in order to do that. Once you have the patched image, just start it in Dolphin. For Dolphin users there's a 1 week wait time after the first connection attempt, so if you get error code 23899 or 23887 after your first connection you will need to wait a week to be able to play.
