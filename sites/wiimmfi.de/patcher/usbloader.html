USB Loaders

Overview

USB Loaders allow you to boot game copies from a USB drive. When you're using USB Loaders you can either manually patch your game images with the ISO patcher, or you use the Wiimmfi patch that's built into the USB Loaders.

Setup

Currently, only the USB Loader GX and WiiFlow Lite have included Wiimmfi patchers. Make sure you're using an up-to-date version of the USB Loader, then open the game (or global) settings and search for the "Private Server" option. By default that'll be set to "Off". Change that setting to "Wiimmfi" and save the settings.

When you now start games through the USB Loader, they will automatically be Wiimmfi-patched.
