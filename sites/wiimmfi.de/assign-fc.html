Assign a friend code

Here you can assign a friend code to an old profile created by Nintendo. It is necessary, if you tried to connect with an old profile and get error 60000.

Infos about the profile:

Console FC:0000-0000-0000-0000
Console Info:Wii
Game:Wii Speak Channel
Last login try:0001

If you are ready to assign a friend code, do the following steps:

  1. Enter your old friend code here.
  2. Submit the form and read the answer.
  3. Connect your profile with Wiimmfi within a time limit of 10 minutes.
  4. Wait until the game shows the connection success or an error message.
